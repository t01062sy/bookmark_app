# 非機能要件 & SLO

## 1. パフォーマンス
- 保存API応答時間 P95 < 1.2秒（正常回線）
- 保存→要約完了 P95 < 20秒

## 2. 可用性
- API稼働率：99.9%/月
- RPO：< 24h（バックアップ日次）
- RTO：< 1h

## 3. セキュリティ
- APIキーはサーバのみ保持
- 外向き通信は許可リスト制限
- 保存データはKMS暗号化（AES-256）
- TLS1.2+通信

- 時間当たり上限：¥300
- 閾値超過時は自動ブレーカー発動
